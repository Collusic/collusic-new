# COLLUSIC - NEW

#### 여러 작곡가들이 음원을 붙여 나가는 릴레이 작곡 웹 서비스

놀면 뭐하니? [유플래쉬](https://namu.wiki/w/%EB%86%80%EB%A9%B4%20%EB%AD%90%ED%95%98%EB%8B%88%3F/%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8%20%EA%B5%AC%EC%84%B1/%EC%9C%A0%ED%94%8C%EB%9E%98%EC%89%AC)에서 영감을 받아 시작하게 된 웹 서비스

내가 만든 음원 위에 다른 작곡가들의 음원을 붙여 하나의 릴레이의 곡을 만들고 싶다?

그럼 콜류직 - 뉴 만한 것이 또 없을테죠.

누군가의 음원 위에 당신의 음원을 이어주세요.

🎤🎶🎸🎹
